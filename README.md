# 💇‍♀️ Peluquería-SASS

Proyecto de práctica centrado en el uso de **SASS** para estilizar una landing page de una peluquería.  
El objetivo principal es explorar y aplicar características avanzadas de SASS como:

- Variables
- Mixins
- Anidamiento
- Funciones personalizadas
- Uso de `@import` y `@use`
- Organización modular de estilos

Este proyecto sirve como ejemplo práctico para quienes desean aprender a trabajar con SASS en entornos reales, manteniendo el código limpio, reutilizable y escalable.

## 🚀 Tecnologías utilizadas

- HTML5
- SASS (SCSS)
- Node.js (para la compilación de SASS)

---

Notas
El proyecto está configurado con un script de npm que compila los archivos SCSS automáticamente al cambiar.
Ideal para practicar buenas prácticas en la organización de archivos SASS y estructuras modulares.
